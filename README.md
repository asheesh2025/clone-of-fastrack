# fastrack

  This is an e-commerce website where customers can purchase products which are shown on the website.
  This is an individual project which was created in 5 days.
    
## Features
    Login Page 
    Signup Page 
    Home Page
    Header
    Footer
    Products Page
    Carts Page 
    
    
## Tools 
    HTML
    CSS
    DOM
    JavaScript
